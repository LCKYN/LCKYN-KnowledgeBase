# CopilotKit

## Overview
CopilotKit is an enterprise-ready agentic application framework that enables developers to build AI copilots into their applications. It provides the infrastructure to connect any LLM or agent framework to user-facing applications through the AG-UI protocol.

## Core Value Proposition
- **Framework-agnostic**: Works with LangGraph, CrewAI, Vercel AI SDK, LlamaIndex, AG2, Mastra, and more
- **LLM-agnostic**: Supports OpenAI, Anthropic, Gemini, Grok, Ollama, and others
- **Enterprise-ready**: Production-grade with governance, analytics, and safety features

## Installation
```bash
npx copilotkit@latest init
```

## The Agentic Application Stack

### 1. User-Facing Applications
CopilotKit sits at the application layer, bringing AG-UI-compatible agents into user-facing interfaces.

### 2. Agent Framework Support
Compatible with major agent frameworks:
- **Vercel AI SDK**: Direct integration
- **LangGraph**: LangChain's graph-based orchestration
- **CrewAI Flows**: Multi-agent workflows
- **LlamaIndex**: Data framework integration
- **AG2 (AutoGen)**: Microsoft's agent framework
- **Mastra**: Agent orchestration
- **Pydantic AI**: Type-safe agent framework
- **Google ADK**: Agent Development Kit
- **AWS Strands**: Amazon's agent framework
- **Microsoft Agent Framework**: Enterprise agent platform
- **A2A Protocol**: Agent-to-Agent communication
- **MCP**: Model Context Protocol

### 3. LLM Integration
Supports all major LLM providers without lock-in.

## Architecture Components

### AG-UI Protocol Integration
CopilotKit is the reference implementation of [[11.06 AG-UI Protocol|AG-UI]], providing:
- Bi-directional agent ↔ user communication
- Shared state management
- Tool-based generative UI
- Subgraph coordination
- Human-in-the-loop workflows
- Agentic chat interfaces

### Client Libraries
- **React AG-UI Client**: First-party support
- **Angular AG-UI Client**: First-party support
- **Community clients**: Emerging support for Golang, Rust, Java

```typescript
// React AG-UI Client
import { useAGUI } from '@copilotkit/react'

const MyComponent = () => {
  const { agent, ui } = useAGUI()
  // Component logic
}
```
