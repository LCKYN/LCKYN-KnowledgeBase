# StrataScratch

**StrataScratch** is a data science interview preparation platform offering real interview questions, portfolio projects, and code validation for landing data roles.

## Key Features

| Feature | Description |
|---------|-------------|
| Real interview questions | Practice actual questions from top companies |
| Code validation | Instant feedback on SQL/Python solutions |
| Portfolio projects | Build demonstrable work for job applications |
| Company-specific prep | Questions tagged by company origin |

## Use Cases

- **Interview prep** - Practice SQL and Python coding challenges
- **Skill assessment** - Validate data manipulation abilities
- **Portfolio building** - Create projects to showcase in applications
- **Company targeting** - Focus on specific company question styles
