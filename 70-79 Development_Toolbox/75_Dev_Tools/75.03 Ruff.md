# Ruff

**Ruff** is an extremely fast Python linter and formatter written in Rust, designed to replace Flake8, Black, isort, and other tools.

## Key Features

| Feature | Description |
|---------|-------------|
| Speed | 10-100x faster than existing tools |
| All-in-one | Linting + formatting + import sorting |
| Drop-in replacement | Compatible with Flake8, Black, isort |
| Autofix | Automatically fixes many issues |
| pyproject.toml | Single config file |

## Installation

```bash
pip install ruff
# or
uv add ruff --dev
```

## CLI Commands

| Command | Purpose |
|---------|---------|
| `ruff check .` | Lint all Python files |
| `ruff check . --fix` | Lint and auto-fix |
| `ruff format .` | Format files (like Black) |
| `ruff check --select I --fix` | Sort imports only |

## Configuration (`pyproject.toml`)

```toml
[tool.ruff]
line-length = 88
target-version = "py311"

[tool.ruff.lint]
select = ["E", "F", "I", "UP", "B"]
ignore = ["E501"]

[tool.ruff.format]
quote-style = "double"
```

## Rule Categories

| Code | Category | Examples |
|------|----------|----------|
| `E` | pycodestyle errors | Whitespace, indentation |
| `F` | Pyflakes | Unused imports, undefined names |
| `I` | isort | Import sorting |
| `UP` | pyupgrade | Python upgrade suggestions |
| `B` | flake8-bugbear | Bug-prone patterns |
| `S` | flake8-bandit | Security issues |

> [!TIP] Recommended Starter Config
> Start with `select = ["E", "F", "I", "UP", "B"]` for a balanced linting experience.

## VS Code Integration

1. Install **Ruff** extension (`charliermarsh.ruff`)
2. Add to `settings.json`:
```json
{
  "[python]": {
    "editor.defaultFormatter": "charliermarsh.ruff",
    "editor.formatOnSave": true,
    "editor.codeActionsOnSave": {
      "source.fixAll.ruff": "explicit",
      "source.organizeImports.ruff": "explicit"
    }
  }
}
```

## Pre-commit Hook

```yaml
repos:
  - repo: https://github.com/astral-sh/ruff-pre-commit
    rev: v0.4.0
    hooks:
      - id: ruff
        args: [--fix]
      - id: ruff-format
```

## Comparison

| Tool | Replaced By Ruff |
|------|------------------|
| Flake8 | `ruff check` |
| Black | `ruff format` |
| isort | `ruff check --select I` |
| pyupgrade | `ruff check --select UP` |
| autoflake | `ruff check --select F` |
