# VS Code

**Visual Studio Code** is a free, open-source code editor by Microsoft with extensive extension ecosystem and built-in Git support.

## Key Features

| Feature | Description |
|---------|-------------|
| IntelliSense | Smart code completion, parameter hints |
| Integrated terminal | Multiple shells within editor |
| Built-in Git | Source control UI, diff viewer |
| Extensions | Massive marketplace ecosystem |
| Remote development | SSH, Containers, WSL support |
| Multi-cursor editing | Edit multiple lines simultaneously |

## Essential Shortcuts

| Action | Mac | Windows/Linux |
|--------|-----|---------------|
| Command Palette | `Cmd+Shift+P` | `Ctrl+Shift+P` |
| Quick Open file | `Cmd+P` | `Ctrl+P` |
| Toggle terminal | `` Ctrl+` `` | `` Ctrl+` `` |
| Go to definition | `F12` | `F12` |
| Find in files | `Cmd+Shift+F` | `Ctrl+Shift+F` |
| Multi-cursor | `Opt+Click` | `Alt+Click` |
| Rename symbol | `F2` | `F2` |

## Useful Settings

> [!TIP] Recommended `settings.json`
> - `"editor.formatOnSave": true`
> - `"editor.bracketPairColorization.enabled": true`
> - `"files.autoSave": "afterDelay"`
> - `"editor.minimap.enabled": false`

## Workspace Configuration

| File | Purpose |
|------|---------|
| `.vscode/settings.json` | Project-specific settings |
| `.vscode/launch.json` | Debug configurations |
| `.vscode/tasks.json` | Build/run tasks |
