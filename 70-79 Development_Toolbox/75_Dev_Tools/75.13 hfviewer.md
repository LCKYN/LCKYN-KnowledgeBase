# hfviewer

## Overview
**hfviewer** is a free web tool by [Embedl](https://embedl.com/) that renders any Hugging Face model's architecture as an **interactive graph**. Paste a repo name or HF URL and instantly explore layers, blocks, and connections without reading config files.

## Key Features

| Feature | Description |
|---|---|
| **Interactive graph** | Pan, zoom, and explore the full model architecture |
| **Family compare** | View multiple model variants side by side (e.g., Gemma 4 lineup) |
| **URL magic** | Replace `huggingface.co` with `hfviewer.com` in any model URL |
| **Model card embed** | Embed the architecture graph directly in a HF model card README |
| **Broad model support** | Works with LLMs, vision models (ViT, YOLO), encoders, encoder-decoders |

## Usage

```
# Direct URL replacement
https://huggingface.co/openai/clip-vit-base-patch32
→ https://hfviewer.com/openai/clip-vit-base-patch32

# Paste repo name
hfviewer.com → paste "mistralai/Mistral-7B-v0.1"
```

> [!TIP] Embed in model cards
> Add the hfviewer embed to your HF model card README so users see the architecture inline. See [hfviewer.com/model-card-embed](https://hfviewer.com/model-card-embed).

## Use Cases
- **Architecture exploration** — quickly understand unfamiliar model structures
- **Debugging** — verify layer counts, block types, and connectivity
- **Documentation** — embed clean architecture visuals in model cards or reports
- **Comparison** — diff two model variants visually with family compare
