# Typst

## Overview
**Typst** is a modern, **markup-based typesetting system** designed as a faster, more approachable alternative to LaTeX. It compiles to PDF with **instant preview**, supports scripting with a built-in language, and handles complex layouts, math, and bibliographies out of the box.

## Scripting

```typst
// Variables
#let title = "ML Report"
#let accuracy = 94.2

// Conditionals
#if accuracy > 90 [
  Model passed threshold.
] else [
  Model needs retraining.
]

// Loop
#for metric in ("Precision", "Recall", "F1") [
  - #metric
]

// Functions
#let highlight(body) = text(fill: red, weight: "bold", body)
This is #highlight[important].
```

> [!TIP] When to choose Typst over LaTeX
> Use Typst for **new documents** where you want fast iteration, clean syntax, and built-in scripting. Use LaTeX when your journal/conference **requires** `.tex` submission or you need a specific LaTeX package.
