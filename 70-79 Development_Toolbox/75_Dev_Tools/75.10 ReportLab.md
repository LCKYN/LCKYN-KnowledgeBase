# ReportLab

## Overview
**ReportLab** is a Python library for **programmatic PDF generation**. It provides low-level canvas drawing and a high-level `Platypus` (Page Layout and Typography Using Scripts) framework for building complex, data-driven PDF documents â€” invoices, reports, certificates, and dashboards.

ðŸ”— **Docs**: [docs.reportlab.com](https://docs.reportlab.com/)

## Two API Levels

| Level | Module | Best For |
|---|---|---|
| **Low-level Canvas** | `reportlab.pdfgen.canvas` | Precise positioning, drawings, custom layouts |
| **High-level Platypus** | `reportlab.platypus` | Flowing documents with paragraphs, tables, images |

## Quick Start

```python
from reportlab.lib.pagesizes import A4
from reportlab.pdfgen import canvas

c = canvas.Canvas("output.pdf", pagesize=A4)
c.drawString(100, 750, "Hello, ReportLab!")
c.save()
```

## Platypus (Document Templates)

```python
from reportlab.platypus import SimpleDocTemplate, Paragraph, Table
from reportlab.lib.styles import getSampleStyleSheet

doc = SimpleDocTemplate("report.pdf")
styles = getSampleStyleSheet()
elements = [
    Paragraph("Monthly Report", styles['Title']),
    Table([["Metric", "Value"], ["Accuracy", "94.2%"]]),
]
doc.build(elements)
```

## Key Components

| Component | Purpose |
|---|---|
| `Canvas` | Low-level: draw text, shapes, images at exact coordinates |
| `SimpleDocTemplate` | High-level: auto page breaks, headers/footers |
| `Paragraph` | Rich text with HTML-like markup |
| `Table` | Tabular data with cell styling, spans, colors |
| `Image` | Embed PNG, JPEG, SVG |
| `Spacer` / `PageBreak` | Layout flow control |
| `TableStyle` | Cell-level formatting (background, borders, alignment) |

## Table Styling Example

```python
from reportlab.lib import colors
from reportlab.platypus import Table, TableStyle

style = TableStyle([
    ('BACKGROUND', (0, 0), (-1, 0), colors.grey),
    ('TEXTCOLOR', (0, 0), (-1, 0), colors.white),
    ('GRID', (0, 0), (-1, -1), 0.5, colors.black),
])
```
