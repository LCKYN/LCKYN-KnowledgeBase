# GitLens

**GitLens** is a VS Code extension that supercharges Git capabilities with inline blame, rich history exploration, and powerful comparison tools.

## Key Features

| Feature | Description |
|---------|-------------|
| Inline blame | See who changed each line, when, and why |
| File history | Full commit history for any file |
| Line history | Track changes to specific lines |
| Commit graph | Visual branch/commit visualization |
| Comparison tools | Compare branches, commits, files |

## Inline Annotations

> [!INFO] Current Line Blame
> GitLens shows author and commit info at the end of each line as you navigate code—essential for understanding code ownership.

| Annotation | Information |
|------------|-------------|
| Current line | Author, relative time, commit message |
| Code lens | Authors summary above functions/classes |
| Gutter blame | Full file blame in gutter |

## Useful Commands

| Command | Purpose |
|---------|---------|
| `GitLens: Toggle File Blame` | Show/hide inline blame |
| `GitLens: Open File History` | View file's commit history |
| `GitLens: Compare with...` | Compare branches/commits |
| `GitLens: Search Commits` | Search by message, author, SHA |
| `GitLens: Show Commit Graph` | Visual commit graph |

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| Toggle line blame | `Alt+B` |
| Show quick commit details | Hover on blame annotation |
| Open commit on remote | Click commit SHA |

## Sidebar Views

| View | Purpose |
|------|---------|
| Commits | Recent commits with details |
| File History | Changes to current file |
| Branches | Branch management |
| Remotes | Remote repository info |
| Stashes | Manage stashed changes |
| Search & Compare | Find commits, compare refs |

## Settings Recommendations

```json
{
  "gitlens.codeLens.enabled": true,
  "gitlens.currentLine.enabled": true,
  "gitlens.hovers.currentLine.over": "line",
  "gitlens.blame.format": "${author|10} ${date|14-}"
}
```
